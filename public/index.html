BookmarkAPI

Functionalities Supported By This API

  1. Bookmarking
    • Create a Bookmark : .../api/create/bookmark
    • Delete a Bookmark : .../api/remove/bookmark/:bookmarkId
  2. Tag
    • Create a Tag : .../api/create/tag
    • Delete a Tag : .../api/remove/tag/:tagId
    • Add a Tag to a Bookmark : .../api/addtag/:bookmarkId/:tagId
    • Remove a Tag from a Certain Bookmark : .../api/removetag/:bookmarkId/:tagId
  3. Display
    • Retrieve all bookmarks : .../api/bookmarks
    • Retrieve all tags : .../api/tags
